# 캐릭터 특성

* 생존자 (명칭 미정)
  * 우선순위 미정
  
    * 금오전
      * ?카드를 뽑을 때 두개의 카드를 뽑아 원하는 카드 선택 가능
        * 제한 x
    
    * 박지효
      * 잡힐 확률을 낮춰줌
        * 1/6 -> 1/2 (주사위 던질 시, 짝수 or 홀수 선택)
          * 제한 미정
    
    * 쿵기덕
      * 은신처 만드는 스킬
        * 크기 미정
          * 두번 사용 가능
    
    * T.O.P
      * 감시자의 위치를 임의로 변경 가능
        * 기회는 단 한번
      
    * 김지원
      * 부활
        * 횟수 제한(미정)
          * 자신 제외
        
        
* 감시자 (명칭 미정)
  * 이름 만으로 우선순위 결정됨
  
    * 박 인턴
      * 카드를 뽑을 때 두 개의 ?카드를 뽑아서 원하는 카드 선택
        * 제한 x
      
    * 김 대리
      * 움직이는 칸 증가
        * 주사위 두 번 던짐
          * 제한 x
        
    * 아 사장
      * Anu의 앞 글자를 따옴
      * 생존자와 한 칸 떨어져 있어도 공격 가능
        * 횟수 미정
